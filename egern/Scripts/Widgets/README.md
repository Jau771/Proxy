# Egern Widgets

This directory stores Egern widget scripts.

## Transparent Mode

The scripts support an optional Egern widget environment variable:

```text
TRANSPARENT_MODE=true
```

When enabled, the root widget omits `backgroundColor` so iOS transparent widget
mode can render the content instead of showing a blank white panel.
